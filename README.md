# Part 1 — Python Basics & Control Flow

**Theme:** Student Grade Tracker | **Marks:** 25

## How to Run

```bash
python3 part1_grade_tracker.py
```

No external dependencies — pure Python 3.

## Tasks Implemented

| Task | Description | Marks |
|------|-------------|-------|
| 1 | Data Parsing & Profile Cleaning (name/roll/marks cleanup, validation, profile cards) | 5 |
| 2 | Marks Analysis — for/while loops, grade labels, subject stats, live entry simulation | 8 |
| 3 | Class Performance Summary — table, pass/fail count, topper, class average | 7 |
| 4 | String Manipulation — strip, title case, count, replace, split, numbered sentences | 5 |

## Notes

- The marks-entry `while` loop (Task 2) uses **simulated inputs** so the script runs non-interactively.  
  In real use, replace `simulated_inputs` with `input()` calls.
- All outputs are printed to stdout with clear section headers.
